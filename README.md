# Python Network Security and Cryptography Programs

This repository contains beginner to intermediate level Python programs related to:

- Cryptography
- Network Security
- Ethical Hacking basics
- Encryption algorithms
- Security utilities

- Caesar Cipher
- XOR Cipher
- Hashing
- Base64 Encoding
- RSA Algorithm
- AES Encryption
- Diffie Hellman Key Exchange
- Digital Signatures
- Port Scanner
- Packet Sniffer
- Simple Firewall
- WHOIS Lookup

## Technologies

- Python 3
- Socket Programming
- Cryptography Library
- Scapy
