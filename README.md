🔐 Python Network Security and Cryptography Lab

A collection of Python programs implementing classical and modern cryptographic algorithms along with mathematical concepts used in Network Security and Cryptography.

This repository was developed as part of Network Security Laboratory coursework and is intended for educational purposes.

\---

📚 Programs Included

# Classical Cryptography

1\. Caesar Cipher

2\. Playfair Cipher

3\. Hill Cipher

4\. Rail Fence Cipher

# Mathematical Algorithms

5\. Euclidean Algorithm

6\. Extended Euclidean Algorithm

7\. Euler's Totient Function

8\. Chinese Remainder Theorem

9\. Fermat's Theorem

# Modern Cryptography

10\. DES (Data Encryption Standard)

11\. AES (Advanced Encryption Standard)

12\. RC4 Stream Cipher

13\. RSA Algorithm

14\. Diffie Hellman Key Exchange
